<h1 align="center">Silicon PHP</h1>

### About Pulsar

A minimal and powerful PHP framework for building fast, modern HTTP services.

Silicon PHP is a framework designed for developers who want to build scalable, maintainable, and well-structured PHP applications.
Inspired by the simplicity of modern frameworks and solid software engineering practices.
